## 그리디 DP

### 그리디
- 항상 현재 시점에서 가장 최적인 선택
- 각 단계에서 최댓값(또는 최솟값)을 선택
- 항상 전체 최적해를 보장하는 것은 아님


---

### DP

1. 점화식
   - 문제에서 규칙을 찾아서 수식(점화식)으로 표현
   - 예시: 피보나치 수열  
     ```

     dp[i] = dp[i-1] + dp[i-2]

     ```

   
2. 메모이제이션
   - 이미 계산한 값을 저장해서 중복 계산을 줄임
   - 연산 횟수를 줄여서 성능 최적화(메모리적으로 많은 부분을 사용)
   - 예시: 피보나치 수열을 재귀로 구현할 때, 이미 계산한 값이 있으면 재사용하여 시간 복잡도를 줄임
